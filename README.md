# Full Stack Product Management System

A full-stack CRUD application built using FastAPI, React, PostgreSQL, and SQLAlchemy.

## Features
- Add products
- View products
- Delete products
- REST API using FastAPI
- React frontend
- PostgreSQL database

## Tech Stack
- FastAPI
- React.js
- SQLAlchemy
- PostgreSQL
- Axios

## Project Structure
backend/
frontend/

## Installation

### Backend
pip install -r requirements.txt
uvicorn main:app --reload

### Frontend
cd frontend
npm install
npm start
